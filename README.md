Nombre del proyecto: Mi primera cafeteria por Mally Lizeth Cano Foronda para Triple Ten.

Este proyecto busca ofrecer un espacio donde los usuarios pueden encontrar recetas veganas fáciles y rápidas que podrían encontrar en el menú de nuestra cafetería pero que también pueden hacer en la comodidad de su casa. Al mismo tiempo pueden reservar una mesa en nuestra cafetería para visitarnos algún día y disfrutar de nuestros platos mientras leen un buen libro. Finalmente se redireccionan a nuestras redes sociales para que nos conozcan mejor.
